# Sales Data Analysis
## Project Overview
Analyzed a real-world sales dataset using Python to identify trends, top products, and business insights.
## Tools Used
* Python
* Pandas
* Matplotlib
## Key Insights
* Sales vary across months (seasonal trends)
* Top products generate most revenue
* Some categories perform better than others
## Conclusion
This project demonstrates data cleaning, analysis, and visualization skills for real-world decision-making.
